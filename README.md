# Creating a Basic Data Lakehouse
This is a learning project which I followed through from a YouTube video. 
The intention is to learn the basic of creating a Data lakehouse using a medallion layer.
The instructor prepared and provided the necessary static data for this project which I then loaded to Databricks unity catalog. 
Converted the static CSV files to delta then loaded it to the Bronze layer.
Loaded the delta file to the silver layer and then proceed in cleaning the Data using pyspark and SQL.
The clean data is then loaded to the Gold layer for consumption.
I created a basic pipeline to automate a real-world scenario when there are new / updated files.
I understand the concept of the intructions and followed the step-by-step guide from the video
The result is that I was able learn and gain the hands on experience and complete the project.
